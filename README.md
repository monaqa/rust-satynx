# rust-satynx

[satynx](https://github.com/monaqa/satynx) ライブラリで用いるコードを自動生成するための Rust 製 CLI ツール．

## Installation

```
git clone https://github.com/monaqa/rust-satynx
cd rust-satynx
cargo install --path .
```

## Usage

```
cat hoge.py | satynx py
cat fuga.rs | satynx rs
```
